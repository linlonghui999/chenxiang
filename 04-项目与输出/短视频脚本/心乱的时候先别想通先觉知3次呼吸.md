# 心乱的时候，先别想通，先觉知 3 次呼吸

## 1. 受众问题

用户在焦虑、浮躁、想太多的时候，常见反应是继续分析：

- 我为什么这样？
- 到底该怎么办？
- 我要怎样才能马上变好？

但越想越乱，因为注意力继续被念头带着走。

## 2. 核心承诺

给用户一个可以马上做的 30 秒练习：

```text
不急着想通 → 先觉知 3 次呼吸 → 让注意力回到当下
```

## 4. 证据与边界

### 可说

- [[修行信息库/已有资料/当下觉悟的方法：觉知呼吸]]指出，呼吸是每个人都有的基础入口，觉知呼吸能帮助觉知自己的心。
- 当心紧张、浮躁时，呼吸往往也会急促；当心静下来，呼吸也会更柔和。
- 对新手来说，呼吸觉知比抽象理论更容易开始。

### 不能夸大

- 不说 3 次呼吸能治好焦虑。
- 不说这个练习能替代医疗、心理咨询或系统禅修。
- 不强迫用户“必须平静”，只邀请用户观察。
- 不把走神当失败。

## 5. 60 秒跟练脚本

### 开头钩子，0–6 秒

心乱的时候，先别急着想通。

你越想，念头可能越多。

先跟我做 3 次呼吸。

### 准备，6–12 秒

不用闭眼也可以。

只要把注意力，从脑子里的事情，轻轻带回呼吸。

### 第 1 次呼吸，12–24 秒

吸气。

知道自己正在吸气。

呼气。

知道自己正在呼气。

不要控制它，只是看见它。

### 第 2 次呼吸，24–36 秒

再来一次。

吸气的时候，感受空气进入身体。

呼气的时候，感受身体稍微松一点。

如果走神了，也没关系。

知道自己走神了，再回来。

### 第 3 次呼吸，36–48 秒

最后一次。

吸气，知道吸气。

呼气，知道呼气。

这一刻，你不需要解决整个人生。

你只需要回到这一口气。

### 收束，48–60 秒

这就是最小的修行入口。

不是马上变好，而是先不被念头完全带走。

心乱时，先别想通，先觉知 3 次呼吸。

## 6. 90 秒解释版脚本

心乱的时候，很多人第一反应是继续想：

我为什么会这样？

我到底该怎么办？

我要怎么马上变好？

但有时候你会发现，越想越乱。

因为你不是在解决问题，你是在被念头继续拖着跑。

这个时候，先别急着想通。

先做一个最小动作：觉知 3 次呼吸。

第一口气，吸气的时候，知道自己正在吸气；呼气的时候，知道自己正在呼气。

第二口气，感受空气进入身体，再感受身体随着呼气松一点。

第三口气，如果你走神了，也没关系。

你只要知道“我刚才走神了”，然后再回到呼吸。

这件事很简单，但它有一个关键作用：

它把你的注意力，从脑子里的故事，带回正在发生的身体。

呼吸是每个人都有的入口。

当你紧张、浮躁，呼吸往往也会急促；当你开始觉知它，心就有机会慢一点。

但注意，这不是说 3 次呼吸能解决所有焦虑。

它不是医疗，也不是万能方法。

它只是一个起点。

当你不再完全被念头带着走，你才有空间重新选择下一步。

所以心乱时，先别急着想通。

先觉知 3 次呼吸。

先回来，再处理。
